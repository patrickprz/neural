RNA PREDITIVA APLICADA EM JOGOS

Projeto realizado pelo aluno Patrick P. Rodrigues para a disciplina de estágio do curso de Engenharia de Computação - UNIFTEC, sob a orientação do professor Me. Carlos Alberto Bertotto.

Patrick P. Rodrigues




  O objetivo desse site é o de servir como uma apresentação das funcionalidades presentes no projeto e de qual maneira elas funcionam, juntamente com a API contida neste domínio.
  O papel da API é o de comunicar o jogo feito na engine Unity3D, com a rede neural feita em Python, para isso a API faz o envio de arquivos JSONs entre API/RNA, para isso a API conta com dois modos de operação. No modo de treinamento os JSONs são enviados da engine para a RNA, contendo um histórico de jogadas, esse histórico de jogadas é recebido pela RNA e processado, fazendo com que ela assimile os padrões recebidos, já no modo de previsão a RNA recebe uma sequencia de comandos e devolve o comando previsto, baseado nas entradas recebidas. Isso tudo é feito com requisições http e a maneira de chamar os métodos da API esta descrita a seguir.

Métodos de funcionamento da API:

Prediction: SERVERIP/prediction/json

Training: SERVERIP/training/json
